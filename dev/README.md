# Development and Utilities (`dev/`)

A utility folder mostly meant as a scratchpad for developers contributing specifically to the repo.

## Key Files:
- **`gen_synthetic_data.py`** ([`dev/gen_synthetic_data.py`](./gen_synthetic_data.py)): Logic for formatting synthetic data.
- **`repackage_data_reference.py`** ([`dev/repackage_data_reference.py`](./repackage_data_reference.py)): Dataset repackaging utilities.
- **Project assets**: Logos and other assets.
